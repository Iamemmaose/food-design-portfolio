Food Design Portfolio

A modern **food-themed design portfolio website** that showcases creative UI concepts and responsive frontend design.
Built to demonstrate clean layout structure and frontend development skills.

**Live Demo:** https://iamemmaose.github.io/food-design-portfolio/

About the Project

The **Food Design Portfolio** is a frontend project focused on presenting food-inspired visuals in a structured and modern layout.

It was built to explore:

* Visual hierarchy in UI design
* Layout composition and spacing
* Responsive web design principles
* Clean and minimal aesthetic presentation

This project is ideal for showcasing **design thinking + frontend execution**.

Key Features

* Modern food-themed UI design
* Fully responsive (mobile, tablet, desktop)
* Clean and minimal visual structure
* Smooth section-based navigation
* Lightweight and fast loading
* Strong visual hierarchy for content presentation

Tech Stack

| Technology | Purpose            |
| ---------- | ------------------ |
| HTML5      | Page structure     |
| CSS3       | Styling and layout |
| JavaScript | Interactivity      |

Deployment

This project is deployed using **GitHub Pages**.

Deployment Process:

1. Push project to GitHub repository
2. Open repository settings
3. Navigate to **Pages**
4. Select branch: `main`
5. Save and publish
